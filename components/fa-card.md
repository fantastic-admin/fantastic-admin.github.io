# FaCard 卡片

## Props

| 属性名       | 说明           | 类型     | 默认值 |
| ------------ | -------------- | -------- | ------ |
| title        | 标题           | `string` | -      |
| description  | 描述           | `string` | -      |
| class        | 自定义类名     | `string` | -      |
| headerClass  | 头部自定义类名 | `string` | -      |
| contentClass | 内容自定义类名 | `string` | -      |
| footerClass  | 底部自定义类名 | `string` | -      |

## Slots

| 插槽名  | 说明           |
| ------- | -------------- |
| default | 自定义默认内容 |
| header  | 头部内容       |
| footer  | 底部内容       |
