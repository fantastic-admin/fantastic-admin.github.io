# SvgIcon SVG图标

## Props

| 参数   | 说明                                                               | 类型            | 可选值                       | 默认值 |
| :----- | :----------------------------------------------------------------- | :-------------- | :--------------------------- | :----- |
| name   | svg 文件名 / Iconify 图标名，详细可阅读《[图标](/guide/svg-icon)》 | string          | -                            | -      |
| flip   | 翻转方式，支持水平、垂直以及同时翻转                               | string          | horizontal / vertical / both | -      |
| rotate | 旋转角度                                                           | number          | 0 - 360                      | 0      |
| color  | 颜色                                                               | string          |                              |        |
| size   | 尺寸                                                               | string / number |                              |        |
