# FaBadge 徽章

## Props

| 属性名  | 说明       | 类型                                        | 默认值      |
| ------- | ---------- | ------------------------------------------- | ----------- |
| variant | 徽章类型   | `'default' \| 'secondary' \| 'destructive'` | `'default'` |
| value   | 徽章内容   | `string \| number \| boolean`               | -           |
| class   | 自定义类名 | `string`                                    | -           |
