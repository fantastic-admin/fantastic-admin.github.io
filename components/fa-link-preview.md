## Props

| 属性名    | 说明                                       | 类型      | 默认值  |
| --------- | ------------------------------------------ | --------- | ------- |
| class     | 主元素自定义类名                           | `string`  | -       |
| linkClass | 链接元素自定义类名                         | `string`  | -       |
| width     | 宽度                                       | `number`  | `200`   |
| height    | 高度                                       | `number`  | `150`   |
| isStatic  | 预览图像是静态的还是URL预览                | `boolean` | `false` |
| imageSrc  | 静态预览图（如果 isStatic 为 true 则需要） | `string`  | -       |
| url       | URL预览（如果 isStatic 为 false 则需要）   | `string`  | -       |
