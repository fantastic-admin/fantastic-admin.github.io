# PcasCascader 省市区街道联动

## Props

| 参数     | 说明                                                 | 类型    | 可选值             | 默认值 |
| :------- | :--------------------------------------------------- | :------ | :----------------- | :----- |
| v-model  | 联动数据选中项绑定值                                 | array   | -                  | \[]     |
| type     | 联动类型，`pc` 省市、`pca` 省市区、`pcas` 省市区街道 | string  | pc / pca / pcas    | pca    |
| format   | 绑定数据的格式化类型                                 | string  | code / name / both | code   |
| disabled | 是否禁用                                             | boolean | -                  | false  |
