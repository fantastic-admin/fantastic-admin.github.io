## Props

| 属性名   | 说明                         | 类型     | 默认值 |
| -------- | ---------------------------- | -------- | ------ |
| words    | 设置动画的单词数组           | `string` | -      |
| duration | 翻转到下个单词的动画持续时间 | `number` | `3000` |
| class    | 自定义类名                   | `string` | -      |
