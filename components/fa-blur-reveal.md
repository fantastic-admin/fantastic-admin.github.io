## Props

| 属性名   | 说明                   | 类型     | 默认值 |
| -------- | ---------------------- | -------- | ------ |
| duration | 持续时间               | `number` | `1`    |
| delay    | 延迟时间               | `number` | `1`    |
| blur     | 子组件的模糊量         | `string` | `10px` |
| yOffset  | 入口动画的垂直偏移距离 | `number` | `20`   |
| class    | 自定义类名             | `string` | -      |
