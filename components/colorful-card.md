# ColorfulCard 多彩渐变卡片 <sup class="pro-badge" />

## Props

| 参数       | 说明                                            | 类型   | 可选值 | 默认值    |
| :--------- | :---------------------------------------------- | :----- | :----- | :-------- |
| color-from | 开始颜色                                        | string | -      | '#843cf6' |
| color-to   | 终止颜色                                        | string | -      | '#759bff' |
| header     | 头部标题                                        | string | -      | -         |
| num        | 数字标记                                        | number | -      | -         |
| tip        | 底行提示文字                                    | string | -      | -         |
| icon       | 图标，使用 SvgIcon 组件，与该组件 name 参数相同 | string | -      | -         |
