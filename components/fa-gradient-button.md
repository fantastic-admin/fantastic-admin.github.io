# FaGradientButton 渐变按钮 &#x20;

## Props

| 属性名       | 说明           | 类型       | 默认值       |
| ------------ | -------------- | ---------- | ------------ |
| colors       | 颜色数组       | `string[]` | 彩虹颜色数组 |
| duration     | 持续时间       | `number`   | `2500`       |
| class        | 自定义类名     | `string`   | -            |
| contentClass | 内容自定义类名 | `string`   | -            |

## Slots

| 插槽名  | 说明     |
| ------- | -------- |
| default | 按钮内容 |
