# SearchBar 搜索面板

## Props

| 参数        | 说明             | 类型    | 可选值 | 默认值 |
| :---------- | :--------------- | :------ | :----- | :----- |
| fold        | 是否折叠         | boolean | -      | true   |
| show-toggle | 是否显示切换按钮 | boolean | -      | true   |
| background  | 是否显示背景     | boolean | -      | false  |

## Slots

| name | 说明                                          |
| :--- | :-------------------------------------------- |
| -    | 内容区域的 HTML 内容，作用域参数为 `{ fold }` |

## Events

| 事件名称 | 说明                        | 回调参数   |
| :------- | :-------------------------- | :--------- |
| toggle   | 切换展开/收起状态的事件回调 | 新状态的值 |
