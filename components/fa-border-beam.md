# FaBorderBeam 边框光束 &#x20;

## Props

| 属性名    | 说明                     | 类型     | 默认值    |
| --------- | ------------------------ | -------- | --------- |
| size      | 大小                     | `number` | `200`     |
| duration  | 持续时间                 | `number` | `15000`   |
| delay     | 延迟时间                 | `number` | `0`       |
| anchor    | 锚点，确定其沿边界的位置 | `string` | `90`      |
| border    | 边框宽度                 | `number` | `1`       |
| colorFrom | 起始颜色                 | `string` | `#ffaa40` |
| colorTo   | 结束颜色                 | `string` | `#9c40ff` |
