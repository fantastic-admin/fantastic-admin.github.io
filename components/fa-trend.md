# FaTrend 趋势标记

## Props

| 属性名  | 说明     | 类型             | 默认值  |
| ------- | -------- | ---------------- | ------- |
| value   | 内容     | `string`         | -       |
| type    | 类型     | `'up' \| 'down'` | `up`    |
| prefix  | 前缀     | `string`         | -       |
| suffix  | 后缀     | `string`         | -       |
| reverse | 颜色反转 | `boolean`        | `false` |
