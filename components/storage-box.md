# StorageBox 储存箱 <sup class="pro-badge" />

## Props

| 参数                  | 说明               | 类型           | 可选值 | 默认值   |
| :-------------------- | :----------------- | :------------- | :----- | :------- |
| model-value / v-model | 需要储存的数据     | object / array | -      | -        |
| name                  | 数据存放的命名空间 | string         | -      | -        |
| title                 | 下拉面板标题       | string         | -      | '储存箱' |

## Slots

| name | 说明                     |
| :--- | :----------------------- |
| -    | 触发下拉面板的 HTML 内容 |

## Events

| 事件名称 | 说明                             | 回调参数   |
| :------- | :------------------------------- | :--------- |
| take-out | 点击下拉面板中某条记录的事件回调 | 记录的数据 |
