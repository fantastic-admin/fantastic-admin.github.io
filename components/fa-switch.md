# FaSwitch 开关

## Props

| 属性名              | 说明                                         | 类型      | 默认值  |
| ------------------- | -------------------------------------------- | --------- | ------- |
| modelValue(v-model) | 值                                           | `boolean` | -       |
| disabled            | 是否禁用                                     | `boolean` | `false` |
| onIcon              | 开启状态图标，即 `FaIcon` 组件的 `name` 属性 | `string`  | -       |
| offIcon             | 关闭状态图标，即 `FaIcon` 组件的 `name` 属性 | `string`  | -       |
