# FaStorageBox 储物箱 <Badge type="pro" text="专业版" />

## Props

| 属性名 | 说明               | 类型              | 默认值 |
| ------ | ------------------ | ----------------- | ------ |
| data   | 需要储存的数据     | `object \| any[]` | -      |
| name   | 数据存放的命名空间 | `string`          | -      |
| title  | 下拉面板标题       | `string`          | -      |

## Slots

| 插槽名  | 说明           |
| ------- | -------------- |
| default | 自定义默认内容 |

## Events

| 事件名  | 说明                         | 回调参数                           |
| ------- | ---------------------------- | ---------------------------------- |
| takeOut | 点击下拉面板中某条记录时触发 | `() => { value: object \| any[] }` |
