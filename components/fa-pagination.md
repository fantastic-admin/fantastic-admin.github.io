# FaPagination 分页&#x20;

## Props

| 属性名 | 说明     | 类型     | 默认值 |
| ------ | -------- | -------- | ------ |
| page   | 当前页码 | `number` | -      |
| size   | 每页条数 | `number` | -      |
| total  | 总数     | `number` | -      |

## Events

| 事件名     | 说明           | 参数                     |
| ---------- | -------------- | ------------------------ |
| pageChange | 页码改变时触发 | `(page: number) => void` |
