## Props

| 属性名          | 说明                     | 类型     | 默认值                  |
| --------------- | ------------------------ | -------- | ----------------------- |
| gradientSize    | 聚光灯效果的半径         | `number` | `200`                   |
| gradientColor   | 聚光灯渐变的颜色         | `string` | `'hsl(var(--primary))'` |
| gradientOpacity | 聚光灯渐变效果的不透明度 | `number` | `0.1`                   |
| class           | 自定义类名               | `string` | -                       |
| slotClass       | 插槽的父容器自定义类名   | `string` | -                       |

## Slots

| 插槽名  | 说明           |
| ------- | -------------- |
| default | 自定义默认内容 |
