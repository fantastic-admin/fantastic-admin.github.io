# FaInteractiveButton 交互按钮

带有悬停动画效果的交互式按钮，鼠标悬停时显示箭头和文字位移。

## 使用场景

* 带有悬停动画效果的交互式按钮，鼠标悬停时显示箭头和文字位移。
* 常见用法：自定义样式、不同尺寸、不同颜色主题、组合使用。

## Props

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| text | 按钮文字 | `string` | `undefined` |
| class | 自定义类名 | `HTMLAttributes['class']` | `undefined` |

## Slots

| 名称 | 说明 |
|------|------|
| - | 无插槽 |

## 注意事项

* 涉及动画、预览或状态反馈时，建议结合实际页面密度控制同屏实例数量与刷新频率。
