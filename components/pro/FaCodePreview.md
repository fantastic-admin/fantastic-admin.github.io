# FaCodePreview 代码预览

代码预览组件，展示可切换源码视图的示例代码。

## 使用场景

* 代码预览组件，展示可切换源码视图的示例代码。
* 常见用法：带标题、组件文档页面、多行源码导入、配合 Markdown 使用。

## Props

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| code | 要显示的源码 | `string` | `required` |
| title | 标题 | `string` | `undefined` |

## Slots

| 名称 | 说明 |
|------|------|
| default | 默认插槽，放置示例组件 |

## 注意事项

* 涉及动画、预览或状态反馈时，建议结合实际页面密度控制同屏实例数量与刷新频率。
