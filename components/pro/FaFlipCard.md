# FaFlipCard 翻转卡片

支持 3D 翻转效果的卡片组件，鼠标悬停时翻转显示背面内容。

## 使用场景

* 支持 3D 翻转效果的卡片组件，鼠标悬停时翻转显示背面内容。
* 常见用法：水平翻转、产品卡片、团队成员卡片。

## Props

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| rotate | 翻转方向 | `'x' \| 'y'` | `'y'` |
| class | 自定义类名 | `HTMLAttributes['class']` | `undefined` |

## Slots

| 名称 | 说明 |
|------|------|
| default | 默认插槽，卡片正面内容 |
| back | 背面内容插槽 |

## 注意事项

* 涉及动画、预览或状态反馈时，建议结合实际页面密度控制同屏实例数量与刷新频率。
