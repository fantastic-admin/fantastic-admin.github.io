# FaSmoothSwipe 平滑滑动

支持平滑切换动画的内容滑动组件，适用于选项卡、步骤切换等场景。

## 使用场景

* 支持平滑切换动画的内容滑动组件，适用于选项卡、步骤切换等场景。
* 常见用法：选项卡切换、步骤表单、图片展示。

## Props

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| activeIndex | 当前激活的索引 | `number` | `required` |

## Slots

| 名称 | 说明 |
|------|------|
| default | 默认插槽，放置多个内容块 |

## 注意事项

* 涉及动画、预览或状态反馈时，建议结合实际页面密度控制同屏实例数量与刷新频率。
