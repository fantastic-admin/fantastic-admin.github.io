# FaSparklesText 闪烁文字 &#x20;

## Props

| 属性名        | 说明         | 类型                                | 默认值                                    |
| ------------- | ------------ | ----------------------------------- | ----------------------------------------- |
| text          | 文字内容     | `string`                            | -                                         |
| sparklesCount | 闪烁星星数量 | `number`                            | `10`                                      |
| colors        | 闪烁颜色     | `{ first: string, second: string }` | `{ first: '#9E7AFF', second: '#FE8BBB' }` |
| class         | 自定义类名   | `string`                            | -                                         |
