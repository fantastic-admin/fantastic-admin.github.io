# FaDigitalCard 数字卡片 <Badge type="pro" text="专业版" />

## Props

| 属性名      | 说明                                 | 类型                         | 默认值 |
| ----------- | ------------------------------------ | ---------------------------- | ------ |
| title       | 标题                                 | `string`                     | -      |
| titleTips   | 标题提示                             | `string`                     | -      |
| icon        | 图标，即 `FaIcon` 组件的 `name` 属性 | `string`                     | -      |
| digital     | 数字                                 | `string \| number`           | -      |
| description | 描述                                 | `string`                     | -      |
| trend       | 趋势                                 | `'up' \| 'stable' \| 'down'` | -      |
| class       | 自定义类名                           | `string`                     | -      |
