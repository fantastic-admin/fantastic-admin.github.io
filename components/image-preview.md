# ImagePreview 图片预览&#x20;

:::danger 注意
该组件已弃用，请使用 `FaImagePreview` 组件。
:::

## Props

| 参数   | 说明     | 类型   | 可选值 | 默认值 |
| :----- | :------- | :----- | :----- | :----- |
| src    | 图片链接 | string | -      | -      |
| width  | 展示宽度 | string | -      | -      |
| height | 展示高度 | string | -      | -      |
