# Select 选择器

## Props

| 属性名              | 说明     | 类型                                                     | 默认值  |
| ------------------- | -------- | -------------------------------------------------------- | ------- |
| modelValue(v-model) | 值       | `string`                                                 | -       |
| disabled            | 是否禁用 | `boolean`                                                | `false` |
| options             | 选项     | `{ label: string, value: string, disabled?: boolean }[]` | -       |
