# AuthAll 鉴权

## Props

| 参数  | 说明                                                                | 类型           | 可选值 | 默认值 |

| value | 权限数据，如果传入为 array 时，全部匹配则鉴权通过，并显示 slot 内容 | array / string | -      | -      |
