# FaTextHighlight 文字高亮 &#x20;

## Props

| 属性名       | 说明             | 类型     | 默认值    |
| ------------ | ---------------- | -------- | --------- |
| delay        | 延迟时间         | `number` | `0`       |
| duration     | 持续时间         | `number` | `2000`    |
| textEndColor | 动画末尾文本颜色 | `string` | `inherit` |
| class        | 自定义类名       | `string` | -         |

## Slots

| 插槽名  | 说明           |
| ------- | -------------- |
| default | 自定义默认内容 |
