# FaPageMain 内容块

## Props

| 属性名   | 说明                                     | 类型      | 默认值  |
| -------- | ---------------------------------------- | --------- | ------- |
| title    | 标题                                     | `string`  | -       |
| collaspe | 是否折叠                                 | `boolean` | `false` |
| height   | 展示高度，当 `collaspe` 为 `true` 时生效 | `string`  | -       |

## Slots

| 插槽名  | 说明           |
| ------- | -------------- |
| default | 自定义默认内容 |
| title   | 标题区内容     |
