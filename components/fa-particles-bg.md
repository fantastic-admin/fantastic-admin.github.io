# FaParticlesBg 粒子背景 &#x20;

## Props

| 属性名    | 说明                                                   | 类型     | 默认值 |
| --------- | ------------------------------------------------------ | -------- | ------ |
| color     | 粒子颜色，支持 HEX 颜色代码                            | `string` | `#FFF` |
| quantity  | 粒子数量                                               | `number` | `100`  |
| staticity | 根据鼠标的接近程度确定粒子移动的量，数值越大，移动越慢 | `number` | `50`   |
| ease      | 粒子移动的缓和效果，数值越小，会使粒子更紧密地跟随鼠标 | `number` | `50`   |
| class     | 自定义类名                                             | `string` | -      |
