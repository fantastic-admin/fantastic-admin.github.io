# PageHeader 页头

## Props

| 参数    | 说明 | 类型   | 可选值 | 默认值 |
| :------ | :--- | :----- | :----- | :----- |
| title   | 标题 | string | -      | -      |
| content | 内容 | string | -      | -      |

## Slots

| name    | 说明                 |
| :------ | :------------------- |
| -       | 右侧区域的 HTML 内容 |
| content | 内容区域的 HTML 内容 |
