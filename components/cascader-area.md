# CascaderArea 省市级联动

## Props

| 参数     | 说明                                                   | 类型    | 可选值 | 默认值 |
| :------- | :----------------------------------------------------- | :------ | :----- | :----- |
| value    | 三级联动中文数组，例如：`['河北省', '唐山市', '市辖区']` | array   | -      | []     |
| size     | 尺寸                                                   | string  | -      | -      |
| disabled | 是否禁用                                               | boolean | -      | false  |
