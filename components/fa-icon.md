# FaIcon 图标

## Props

| 属性名 | 说明                                                                                               | 类型                                   | 默认值 |
| ------ | -------------------------------------------------------------------------------------------------- | -------------------------------------- | ------ |
| name   | 图标名称，支持 `本地 svg 文件` / `Iconify 图标` / `UnoCSS 图标`，详细可阅读《[图标](/guide/icon)》 | `string`                               | -      |
| flip   | 翻转方式                                                                                           | `'horizontal' \| 'vertical' \| 'both'` | -      |
| rotate | 旋转角度                                                                                           | `number`                               | 0      |
| color  | 颜色                                                                                               | `string`                               | -      |
| size   | 尺寸                                                                                               | `string \| number`                     | -      |
