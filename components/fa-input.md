# FaInput 输入框

## Props

| 属性名              | 说明       | 类型      | 默认值 |
| ------------------- | ---------- | --------- | ------ |
| modelValue(v-model) | 输入框值   | `string`  | -      |
| disabled            | 是否禁用   | `boolean` | false  |
| class               | 自定义类名 | `string`  | -      |
