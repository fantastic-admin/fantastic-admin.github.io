### 购买流程

1. 请先加作者微信进行在线咨询。

2) 通过**微信**或**支付宝**扫码支付。

3. 支付成功后需提供：
   * **付款截图凭证**
   * **授权邮箱号（常用邮箱）**。专业版客户的唯一凭证，用于接收框架相关通知公告的唯一方式，但通常情况下我们不会随意打扰你。
   * **Github 用户名**。专业版源码托管在 Github 平台，需提供你的 Github 用户名，我们会邀请你加入组织。
4. 同意邀请后，进入[专业版开发者私有组织](https://github.com/fantastic-admin)，获取源码。
